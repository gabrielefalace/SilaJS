Sila JS

smart conditions and matchers

This small JavaScript library enable writing more readable conditions and selector functions, especially useful when dealing with collections (e.g. processing JSON). Both basic condition checkers and related function generators (to use with filter) are provided.

It's pure JavaScript, apply anywhere in any project!

Basically you will replace this pretty cluttered code

if (x === 3 || x === 8 || x === 12) {
    var result = list.filter(function(element) {
        return element > 3 && element < 25;
    });
}

with this

if (value(x).isIn(3, 8, 12)) {
    var result = list.filter(matching.between(3, 25));
}
